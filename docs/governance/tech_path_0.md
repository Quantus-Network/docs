# TechPath 0: Technical Collective Root Track

## Parameters

- **Name**: root
- **Maximum Deciding**: 1 referendum at a time
- **Decision Deposit**: 1,000 QUAN
- **Prepare Period**: 1 day
- **Decision Period**: 5 days
- **Confirm Period**: 2 days
- **Minimum Enactment Period**: 2 days

## Approval and Support Thresholds

### Minimum Approval
- Linear decreasing curve from 100% to 75%
- Requires 100% approval at the beginning of the voting period
- Requires at least 75% approval at the end

### Minimum Support
- Disabled (0%)
- Support parameter is not used in this track
- Focus is on high approval requirements from Technical Collective members

## Permissions

- Reserved for critical protocol changes and upgrades
- Only accessible by Technical Collective members
- Highest security thresholds
- Used for fundamental system modifications
- Requires near-unanimous approval from technical experts
